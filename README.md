# Code4Community Marketing Site

A standalone Next.js marketing website for [Code4Community](https://code4community.net) — advertising custom software development services for nonprofits and community organizations.

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Structure

- `app/` — Next.js App Router pages and layout
- `components/` — Marketing sections (Hero, Services, Work, etc.)
- `lib/constants.js` — Content and links to the main platform
- `public/` — Logo, partner logos, and portfolio screenshots

CTAs link to the main platform at `https://code4community.net` for signup, contact, and the full portfolio.
